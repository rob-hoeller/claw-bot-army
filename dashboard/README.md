## Overview

Admin dashboard for managing the HBx Agent Force. Built with Next.js, Supabase Auth, and Tailwind CSS.

## Setup

1. `cd dashboard`
2. Copy `.env.example` to `.env.local` and fill in Supabase credentials
3. `npm install`
4. `npm run dev`

## Deployment

Connect to Vercel, point to `/dashboard` directory.
